Type Definitions

override

An override entry. Overrides can override three things: properties, methods, and namespaces. These follow the way that objects/namespaces/etc are defined in api.jsca, even though namespaces really are just properties with objects. Each entry must only have one of value, callFunction , or obj, and the type of override is determined by which property is present.
Type:
  • Object
Properties:
Name Type Argument Description
regex RegExp The regex defining the API or APIs that are being overridden
value module:base.BaseType <optional>
 The value of the property
callFunction module:plugins/TiApiProvider.overrideFunctionCallback <optional>
 The Function.Call implementation for this function property
obj module:base.BaseType <optional>
 The value of the object
