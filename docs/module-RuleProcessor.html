Type Definitions

returnTuple

A return tuple is a value returned from a JavaScript statement. It is always composed of three elements. The first is a string and represents the status. Typically this is "normal", but can also be things like "throw" or "return". The second element is a primary value for the statement, if any, such as the value returned from a function when the first element is "return". The third element is used for miscellaneous purposes.
