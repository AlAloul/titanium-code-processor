<static> arrayGen(headings, entries, entriesOrder) → {string}

Generates a table that plays nicely with mono-spaced fonts (e.g. like those found in a terminal). It works by generating a single row for each object in the entries array. Each column in the row is pulled from the object using the corresponding key in the entriesOrder array.
Parameters:
