<static> pluralize(singular, plural, value) → {string}

Creates a properly pluralized version of a string given a numerical value. Input strings can use "%s" to denote where the value to pluralize should be placed in a sentence (multiple "%s" instances are supported). The correct string is selected based on whether the value is 1 or not.
Parameters:
